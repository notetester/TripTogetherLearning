# 담당별 보기 — 태그 필터

> 페이지마다 **담당(A~D)·도메인** 태그가 붙어 있습니다. 아래에서 담당이나 도메인을 골라, **자기 범위 챕터를 한 번에 묶어서** 보세요.

::: tip 담당 묶음이란
실제 팀 분담을 **익명 라벨(담당 A~D)**로만 표시합니다. 각 라벨이 가리키는 도메인 묶음은 칩에 적혀 있으니, 자기 범위에 해당하는 라벨을 누르면 됩니다. (특정 사람을 지칭하지 않는 중립 표기)
:::
