## 이 사이트의 학습 흐름

TripTogether는 4명이 도메인을 나눠 만든 **여행 올인원 플랫폼**(Spring Boot 4 · Java 21 · MyBatis · MySQL · JSP)입니다.
이 사이트는 외우는 자료가 아니라 **말하기 훈련 자료**입니다. 각 페이지는 같은 순서를 따릅니다.

```text
1. 한 줄 정의      →  2. 왜 이렇게 설계      →  3. 어떤 기술로 구현
4. 동작 원리       →  5. 구현 상태(됨/계획)  →  6. 면접 답변(3단계)
7. 꼬리질문        →  8. 직접 말해보기       →  ✅ 퀴즈
```
